## What is GO ?

* Go was created at Google in 2007 and released publicly in 2009.
* Go is a statically typed, compiled programming language.
* It is designed for simplicity, speed and efficient concurrecny.
* Go uses goroutines and channels to handle concurrent tasks easily.
* The language has a **built-in garbage collector** for automatic memory management
* Go complies directly to **native machine code**, making programs fast.
* It has a simple syntax and a small standard library that makes it easy to learn
* Go is widely used for **cloud infrastructure , backend services, and DevOPs tools**
* Many poplular tools are written in GO, including Docker , K8s, and Terraform
